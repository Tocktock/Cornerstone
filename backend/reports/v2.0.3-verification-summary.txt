  RUN_POSTGRES_TESTS=1 \
  PERSISTENCE_BACKEND=postgres \
  DATABASE_URL='postgresql+psycopg://cornerstone:cornerstone@localhost:5432/cornerstone' \
  python scripts/run_dependency_complete_verification.py --strict --confirm-live-db

Reason:
  This sandbox does not provide the full dependency-complete environment needed for the strict run. In particular, live PostgreSQL service access and several package dependencies used by strict verification are not available here.

Intended execution environment:
  - Python >=3.13,<3.15
  - pip install -e '.[dev]'
  - PostgreSQL with required extensions
  - DATABASE_URL pointed at a disposable verification database
  - PERSISTENCE_BACKEND=postgres
  - RUN_POSTGRES_TESTS=1

New CI contract:
  .github/workflows/dependency-complete-verification.yml provisions PostgreSQL, installs dev dependencies, renders the plan, runs strict verification, and uploads reports.
