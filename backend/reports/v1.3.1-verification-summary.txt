Cornerstone v1.3.1 verification summary
=======================================

Release goal
------------
Add manual uploaded data ingestion for UTF-8 text-like files and pasted text.
Uploaded data is normalized into existing manual SourceObject records and passed
through the existing sync path to create Artifacts and EvidenceFragments.

Non-goals preserved
-------------------
- No LLM ontology extraction.
- No ConceptCandidate or RelationCandidate runtime model.
- No automatic Concept creation.
- No automatic ConceptRelation creation.
- No ontology graph inference from uploaded data.
- No PDF, Office, image, archive, or arbitrary binary parsing.

Important implementation boundary
---------------------------------

Those EvidenceFragments still require normal review before they can support official ontology objects.
