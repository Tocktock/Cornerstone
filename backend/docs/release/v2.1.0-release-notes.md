# v2.1.0 Release Notes — Live LLM Ontology Provider

`v2.1.0` adds a gated `live_llm` ontology extraction provider.

## Added

- Live provider config: `ONTOLOGY_LIVE_LLM_ENABLED`, API URL, API key, model name, prompt version, fixture JSON, and timeout.
- Strict structured output validation for concepts and relations.
- Evidence-scope validation before candidate persistence.

## Trust boundary

Extractor output remains candidate-only. Human review is still required before official graph mutation.
