# v1.2.1 — Ontology Contract Readiness

## Release summary

`v1.2.1` is a documentation-first release that defines the ontology Single Source of Truth direction for Cornerstone.

It does not change runtime API behavior.

It adds detailed documents for:

```text
- ontology product contract
- ontology domain model proposal
- versioned ontology implementation plan
- official trust-boundary ADR
- future release document template
```

## Why this release exists

The next major product direction introduces LLM-assisted concept and relation extraction. That work must be constrained before implementation.

The contract established by this release is:

```text
LLM output is candidate-only.
Official graph objects require reviewed evidence.
The reviewed ontology graph is the Single Source of Truth.
```

## Product readiness

This release is ready when the team agrees that:

```text
1. Cornerstone's future SSOT is the reviewed ontology graph, not raw source documents.
2. Concepts are graph nodes.
3. ConceptRelations are graph edges.
4. LLM extraction produces candidates only.
5. Human review is required for official graph state.
6. Default graph depth is 1.
7. "Settlement" is the reference example for the first ontology proof.
```
