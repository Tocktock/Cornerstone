# Cornerstone v2.0.0 Release Notes

## Release theme

Ontology SSOT Release.

v2.0.0 stabilizes the backend contract for evidence-grounded, human-reviewed ontology Single Source of Truth serving.

## Added

```text
GET /v1/ontology/ssot/readiness
```

The endpoint returns a read-only readiness checklist for a focus concept.

Added CLI support:

```bash
cornerstone proof run --ssot-readiness
```

Added documents:

```text
docs/47-ontology-ssot-release-v2.0.0.md
docs/release/v2.0.0-ontology-ssot-readiness.md
docs/release/v2.0.0-operator-checklist.md
docs/release/v2.0.0-release-notes.md
```

## Changed

```text
- package version updated to 2.0.0
- release checker updated for v2.0.0
- README current version updated
- API contract updated with SSOT readiness endpoint
- CLI proof report scope includes ssotReadiness
```

## Not changed

```text
- no live external LLM provider
- no automatic candidate approval
- no automatic official graph mutation
- no graph visualization UI
- no graph depth above 1
- no destructive cleanup
```

## Trust boundary

The official reviewed ontology graph remains the Single Source of Truth. Raw source data, connector sync output, extraction output, and pending candidates remain non-official until reviewed through the existing gates.

## Upgrade note

No database migration is required for v2.0.0 because the readiness endpoint is read-only and stores no new records.
