# v2.0.2 Release Notes — Product Documentation Layer

## Summary

`v2.0.2` is a documentation-first release that adds a separate product documentation layer for Cornerstone.

The release answers product questions before technical/release questions:

```text
What is Cornerstone?
What user problem does it solve?
How does the ontology graph work?
Why is the reviewed graph the Single Source of Truth?
How does the settlement example work?
How should operators prove readiness?
```

## Trust boundary

The product trust boundary remains unchanged:

```text
Raw source data is not the SSOT.
Extractor output is not the SSOT.
Pending candidates are not the SSOT.
Reviewed official ontology graph is the SSOT.
```
