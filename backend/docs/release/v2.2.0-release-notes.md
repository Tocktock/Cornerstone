# v2.2.0 Release Notes — Review Operator Experience

`v2.2.0` adds operator-facing review summary and preview APIs plus CLI helpers.

## Added

- `GET /v1/ontology/review-queue/summary`
- `GET /v1/ontology/concept-candidates/{candidateId}/preview`
- `GET /v1/ontology/relation-candidates/{candidateId}/preview`
- `cornerstone review queue`
- `cornerstone review preview`

## Trust boundary

Preview endpoints are read-only. They do not approve, merge, reject, or officialize candidates.
