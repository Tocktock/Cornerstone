# v2.3.0 Release Notes — Graph Visualization Contract

`v2.3.0` makes graph responses easier for a UI or integration consumer to render.

## Added

- Visualization node and edge models.
- Graph-level state legend, layout hints, citation panel, and review provenance panel.
- Empty-state metadata when no official graph is available.

## Trust boundary

The graph service still reads official graph state only for `mode=official`; it does not create inferred graph objects.
