# v2.5.0 Release Notes — External Integration Package

`v2.5.0` packages Cornerstone's official ontology graph and SSOT readiness for external consumers through the `external_integration_package` path.

## Added

- `GET /v1/integration/package/manifest`
- `GET /v1/integration/ontology/{concept}`
- Candidate bypass rejection through `includeCandidates=true`
- Explicit `candidate bypass` guardrail wording for integration consumers

## Trust boundary

The integration package exposes reviewed official graph data, citations, readiness, trust state, and unsupported state. It does not expose candidates as official truth and does not mutate the graph.
