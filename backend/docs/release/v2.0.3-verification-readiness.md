## No product behavior change

`v2.0.3` does not add new runtime behavior. It adds verification machinery for the existing backend contract.

```text
Raw documents are inputs.
Extractor or LLM output is a proposal.
Pending candidates are not official truth.
The reviewed official ontology graph is the Single Source of Truth.
```

## Required strict environment

Strict verification requires:

```text
Python >=3.13,<3.15
pip install -e '.[dev]'
PostgreSQL with required extensions
DATABASE_URL set to a disposable verification database
PERSISTENCE_BACKEND=postgres
RUN_POSTGRES_TESTS=1
```

## Sandbox result

The current sandbox can verify compile/static/plan-only checks, but it does not contain every dependency required for a strict run. The strict run is intentionally represented as a CI/local dependency-complete contract.
