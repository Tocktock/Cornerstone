# 거제시 민생회복지원금 조례 수동 검토 코퍼스

**수집일:** 2026-07-12

**용도:** CornerStone UI에서 한국어 실세계 의사결정 자료를 수동으로 검토하기 위한 탐색용 코퍼스

**상태:** 로컬 수동 검토 입력. VS5의 동결된 영문 25건 평가 코퍼스와 별개이며, 어떤 시나리오의 PASS 근거도 아니다.

이 코퍼스는 2025년 거제시 민생회복지원금 조례 논의를 두 시점으로 나눈다. 원문 HTML은 `raw/`, HTML에서 기계적으로 변환한 전체 텍스트는 `derived/`, 현재 CornerStone UI에 넣을 파일은 `upload/`, 표결 이후 결과는 `held-out/`에 있다.

## UI에서 사용하는 방법

1. CornerStone을 실행한다.
2. 한 번에 한 패킷의 `upload/` 파일만 모두 Drop한다.
3. 아래 의사결정 질문을 사용해 Brief를 만든다.
4. Brief가 완성될 때까지 같은 패킷의 `held-out/` 파일은 열거나 업로드하지 않는다.
5. Brief의 금액, 날짜, 법적 근거, 발언 주체, 불확실성마다 citation을 열어 원문과 대조한다.
6. 마지막에 `held-out/`을 열어 실제 후속 결과와 비교한다. 실제 표결 결과는 정책적으로 옳은 답의 정답표가 아니라 시간 순서와 사실 추적용 검산 자료다.

### Packet A — 2025-06-30 표결 직전

업로드 위치: `packet-a-2025-06-30/upload/`

권장 질문:

> 의안번호 519를 지금 가결, 부결, 또는 보류해야 하는가? 조례 본문이 허용하는 범위와 당시 집행 계획을 구분하고, 찬반 발언자의 주장과 문서로 직접 확인되는 사실을 구분하여 권고안을 작성하라. 아직 확인되지 않은 법률·재정·효과성 쟁점도 명시하라.

검토 포인트:

- 조례 본문이 특정 총액이나 1인당 지급액을 직접 고정하는가?
- 470억 원은 조례 문언, 비용추계, 정치적 주장 중 어디에서 나오는가?
- 보건복지부 협의 필요성에 관한 상반된 주장을 문서만으로 확정할 수 있는가?
- 중앙정부 소비쿠폰과의 중복효과가 독립 자료로 입증되어 있는가?

### Packet B — 2025-09-19 표결 직전

업로드 위치: `packet-b-2025-09-19/upload/`

권장 질문:

> 의안번호 519에서 584로 무엇이 바뀌었으며, 6월에 제기된 반대 사유가 얼마나 해소되었는가? 현재 자료만으로 가결, 부결, 또는 심의보류 중 하나를 권고하고, 확인된 사실·발언자의 주장·미확인 사항을 분리하라.

검토 포인트:

- 10만 원/20만 원 차등 지급과 250억 원 계획이 조례 자체의 구속적 상한인지, 별도 집행 계획인지 구분하는가?
- 0.8% 부채비율, 소비쿠폰 사용률, 업종별 사용 비중을 독립 검증된 사실로 오인하지 않는가?
- 6월안과 9월안을 같은 의안으로 합치지 않고 의안번호와 시점을 구분하는가?
- 의안번호 584의 공식 조례 본문에서 추가된 `한시적 일회성` 문구와 2026년 6월 30일 유효기간을 519 본문과 직접 비교하는가?
- 심사보류안 표결과 본 조례안 표결을 서로 다른 결정으로 구분하는가?

## 파일 정책

- `raw/`는 공식 웹페이지의 다운로드 시점 스냅샷이다.
- `derived/`는 macOS `textutil`로 HTML을 UTF-8 텍스트로 변환하고 non-breaking space와 줄 끝 공백만 정규화했다.
- `upload/`는 위 전체 텍스트에서 해당 안건과 시점에 필요한 연속 구간만 잘라낸 것이다. 문장 재작성이나 사실 보강은 하지 않았다.
- Packet B의 `03_bill-584-official-appendix-text.txt`는 공식 PDF 3-7쪽을 사람이 렌더링 이미지와 대조해 전사한 예외다. 원본 PDF, 공식 URL, 페이지 범위, 해시를 함께 보존하며 줄바꿈 외에는 재작성하지 않았다.
- `held-out/`은 표결 이후 정보이므로 사전 의사결정 Brief 입력에서 제외한다.
- 원문과 변환본의 SHA-256은 각 패킷의 `packet.json`에 기록한다.

## 사용 및 재배포 주의

자료는 국회도서관 지방의정포털과 거제시청에서 공개 열람되는 공식 기록이다. 그러나 이 패킷은 각 사이트의 재배포 허락 범위를 확정하지 않았다. 현재 용도는 로컬·비공개 제품 검토로 제한하며, 코퍼스 자체를 공개 배포하거나 학습 데이터로 전환하기 전에는 원 출처의 최신 이용조건을 별도로 확인한다.
