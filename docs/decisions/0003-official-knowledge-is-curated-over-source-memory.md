# 0003 - Official knowledge is curated over source memory

- **Status:** Accepted

## Context

Raw source memory is necessary but insufficient. The product must distinguish ingested source content from curated official meaning.

## Decision

Artifacts and evidence preserve source memory. Official concepts, relations, decisions, and answers are curated outputs built on top of that memory.

## Consequences

- Source sync does not directly publish official meaning.
- Review and validation remain mandatory for official outputs.
